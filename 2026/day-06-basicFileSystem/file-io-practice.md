### Creating Files and Folders

 - **mkdir -p folderName** (we can use -p flag to specify path to create the folder)
 - **touch file.txt** (To create a file)
 - **echo "hello"** > file.txt (automatically creates file if not exits, overrides text)
 - **echo "world"** >> file.txt (automatically creates file if not exits, overrides text)

###  Editing / File operations

 - **vim/ nano** (To edit the file)
 - **cp source destination** (To copy the file)
 - **mv source destination** (To cut and paste/ Also used to Rename)
 - **cat -b file.txt** (To Read the file)
 - **head -n 5 file.txt** (Read Top 5 Lines)
 - **tail -n 5 file.txt** (Read bottom 5 Lines)
